Node library

Apart from the Atomic nodes, Designer also comes with a library of pre-made Graph Instances, divided up into many categories. There are over 200 different ones, some come in handy very often, some are intended for very specific use cases.This area of the documentation documents and explains all Library nodes and provides pointers and tips on which nodes are good to get started with.

The library panel of Designer displays more categories than we currently have documented, Most notably MDL nodes are missing, these might be added in the future.
