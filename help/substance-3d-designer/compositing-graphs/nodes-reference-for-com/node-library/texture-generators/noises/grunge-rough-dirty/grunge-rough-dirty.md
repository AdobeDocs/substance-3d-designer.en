## Description

The **Grunge Rough Dirty** node generates a grunge map akin to a rough dirty surface with fine parallel scratches.

</td>
</tr>
</table>

## Parameters

* **Balance** *Float*Adjusts the balance between dark and bright values.
* **Contrast** *Float*Adjusts the contrast of the image.
* **Invert** *Boolean*Inverts the output of the image, using a `1-x` operation.
* **Non Square Expansion** *Boolean*Enables compensation of squash and stretch with non-square ratios.
* Advanced
  * **Main Grunge Intensity** *Float*Adjusts the intensity of the main grunge texture used to break up the surface.
  * **Invert Scratches** *Boolean*Inverts the luminance of the scratches on the surface.
  * **Scratches Intensity** *Float*Adjusts the intensity of the scratches on the surface.
  * **Grain Intensity** *Float*Adjusts the intensity of the global grain effect.
