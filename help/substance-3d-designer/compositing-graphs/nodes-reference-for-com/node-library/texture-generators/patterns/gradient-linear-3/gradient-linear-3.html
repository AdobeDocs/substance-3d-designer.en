Description

The most advanced linear gradient. Instead of the rounded, pipe-like profile of Linear Gradient 2, this node returns a sharp, straight slope, plus provides additional control for the midpoint.

Parameters

  • Tiling: 1 - 16
    Sets the amount of times the result should tile.
  • Position: 0.0 - 1.0
    Sets where the midpoint or peak of the gradient lies.
  • Rotation: 0, 90Â°
    Changes the direction from left-right to top-bottom, or vice versa.
