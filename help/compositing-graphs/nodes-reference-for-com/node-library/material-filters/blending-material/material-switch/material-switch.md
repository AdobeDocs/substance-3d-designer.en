## Description

This node is the Multi-Cannel, Full material version of [Switch](../../../filters/blending/switch/switch.md). It takes two materials as input and returns only one of them based on the switch parameter.

## Parameters

### Parameters

* **Channels**  
  Toggle material channels on and off in this group, for example when using Specular/Glossiness maps instead of Metallic/Roughness.
* **Switch**: *False/True*Switch to return either Material 1 or 2.
