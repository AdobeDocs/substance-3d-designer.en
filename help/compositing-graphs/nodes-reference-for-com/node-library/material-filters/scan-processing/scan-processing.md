# Scan Processing

This category is intended to meet your needs for processing photos and scans into full, tiling PBR materials. It contains a large assortment of tools with functions to remove lighting, crop out borders and improve tiling.

The following nodes are good introductions:

* [Clone Patch](../../../../../compositing-graphs/nodes-reference-for-com/node-library/material-filters/scan-processing/clone-patch/clone-patch.md)
* [Multi Crop](../../../../../compositing-graphs/nodes-reference-for-com/node-library/material-filters/scan-processing/multi-crop/multi-crop.md)
* [Smart Auto Tile](../../../../../compositing-graphs/nodes-reference-for-com/node-library/material-filters/scan-processing/smart-auto-tile/smart-auto-tile.md)
