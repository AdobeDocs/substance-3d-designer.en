## Description

A simple contrast and luminosity (brightness) adjustment.

## Parameters

* **Contrast**: *-1.0 - 1.0*  
  Adjusts the contrast of the result.
* **Luminosity**: *-1.0 - 1.0*  
  Adjusts the luminosity (brightness) of the result.
