Histogram Range

Histogram Range

In: Filters/Adjustments

Simple

Description

Reduce and/or move the range of a grayscale input. Can be used to remap transitions, similar to Contrast Luminosity, but with different controls that might make more sense for some situations.
Also see Histogram Scan for another, more useful way to remap the range.

Click here to watch a Substance Academy video on Histogram Range.

Parameters

  • Range: 0.0 - 1.0
    How much to reduce the range down from. This is similar to moving both Levels min and Max sliders inwards.
  • Position: 0.0 - 1.0
    Offset for the range reduction, setting a different midpoint for the range reduction.
