## Description

Clamps input values to defined limits.

## Parameters

* **Min**: *0.0 -* 1.0  
  Lower clamp limit.
* **Max**: *0.0 - 1.0*Upper clamp limit.
* **Apply to Alpha**: *False/True* (Color version only)  
  Choose whether clamping is applied to the alpha as well.
