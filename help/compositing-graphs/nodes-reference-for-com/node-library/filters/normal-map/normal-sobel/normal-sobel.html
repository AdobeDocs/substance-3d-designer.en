Normal Sobel

Normal Sobel

In: Filters/Normal Map

Simple

Description

Converts a Heightmap input to a Normalmap output. A slightly more advanced version of the Normal Atomic Node, this node uses Sobel sampling rather than the standard sampling method.

Parameters

  • Intensity: 0.0 - 3.0
    Strength of the converted normals.
  • Normal Format: OpenGL, DirectX
    Switches between different Normalmap formats (inverts the green channel).
