## Description

Converts an input in Polar Coordinates (Angle &amp; Radius) to Cartesian Coordinates (X &amp; Y). The reverse is possible with [Cartesian to Polar](../cartesian-to-polar/cartesian-to-polar.md).

## Parameters

*No Parameters.*
