Mask Generators

This category contains a selection of black-and-white mask-generating nodes. Based on baked map information, they generate masks that can then be used to blend together materials and other effects. These nodes are similar to Smart Masks and Generators in Substance Painter.

All of these nodes require baked maps, as without baked maps there won't be much of a result.

The main intended usage is to use these mask generators with multi-channels materials. Once a mask is generated, it is used as the blend mask for Material Blend.

Some interesting nodes in this category are:

  • Dripping Rust
  • Edge Damages
  • Bottom To Top
  • Mask Builder
