## Description

The **Grunge Concrete** node generates a grunge map akin to the height map of a concrete surface.

## Parameters

* **Balance** *Float*Adjusts the balance between dark and bright values.
* **Contrast** *Float*Adjusts the contrast of the image.
* **Invert** *Boolean*Inverts the output of the image, using a `1-x` operation.
* **Non Square Expansion** *Boolean*Enables compensation of squash and stretch with non-square ratios.
* Advanced
  * **Base Noisiness** *Float*Adjusts the noisiness of the base texture.
  * **Dirt Specks Opacity** *Float*Adjusts the opacity of the specks of dirt.
  * **Invert Dirt** *Boolean*Inverts the impact of the specks of dirt.
  * **Scratches Opacity** *Float*Adjusts the opacity of the scratches.
  * **Sharpen** *Float*Adjusts the intensity of a sharpening effect applied to the image.
  * **Large Variation Intensity** *Float*Adjusts the large scale (low frequency) variations applied to the base texture.
