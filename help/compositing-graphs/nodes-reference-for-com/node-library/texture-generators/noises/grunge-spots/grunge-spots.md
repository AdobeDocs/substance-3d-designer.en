## Description

The **Grunge Spots** node generates a grunge map akin to fine splattered spots.

</td>

</table>

## Parameters

* **Balance** *Float*Adjusts the balance between dark and bright values.
* **Contrast** *Float*Adjusts the contrast of the image.
* **Invert** *Boolean*Inverts the output of the image, using a `1-x` operation.
* **Non Square Expansion** *Boolean*Enables compensation of squash and stretch with non-square ratios.
* Advanced
  * **Details** *Float*Adjusts how much spots are *warped* and broken up into finer spots.
  * **Coverage** *Float*Adjusts the coverage of the spots in the image.
  * **Coverage Contrast** *Float*Adjusts the contrast of the *mask* used to control  
    the coverage of the spots in the image.
