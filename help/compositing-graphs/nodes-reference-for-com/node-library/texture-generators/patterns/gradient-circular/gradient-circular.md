## Description

Creates a grayscale gradient transition defined by two custom points in a circular fashion. The transition is not from a to b, but rather as a revolution around the first point, starting and ending at the second point. Keep in mind results will never tile.

## Parameters

* **Point 1**:   
  First point to revolve the gradient around, does not have to be centered
* **Point 2**:   
  Second point for gradient to start and end at.
* **Non Square Expansion**: *False/True*  
  Enable compensation of squash and stretch with non-square ratios.
