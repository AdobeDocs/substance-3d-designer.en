## Description

This node generates a simple weave pattern with a few options. It allows for more control than the pre-defined weave patterns and presents a pattern that cannot be achieved with other nodes.

## Parameters

* **Tile X**: *1 - 20*Sets how many blocks repeat on the X-axis.
* **Tile Y**: *1 - 20*Set how many blocks repeat on the Y-axis.
* **Shape**: *0.0 - 1.0*Sets curve height profile of the stitch.
* **Weave**: *1 - 10*Sets how many stitches per block.
* **Gap**: *0.0 - 1.0*Sets the gap between stitches on X- and Y-axis.
* **Non Square Expansion**: *False/True*  
  Enables compensation of squash and stretch with non-square ratios.
