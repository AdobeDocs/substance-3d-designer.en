## Description

Generates a primitive (polygon) in the Paths format.

Use the [Path 2D Transform](../path-2d-transform/path-2d-transform.md) node to position the primitive precisely.

## Output connectors

<b>Paths</b> *Color*  
Contains a list of one encoded path, describing a list of encoded segments.  
This is not indended to be used or modified directly. Search for Paths to find compatible nodes.
