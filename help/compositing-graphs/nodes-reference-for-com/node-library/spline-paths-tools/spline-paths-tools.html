Spline & Path Tools

Spline & Path Tools are a collection of nodes for working with paths and splines in Substance graphs:

Paths are a series of points connected into straight lines.

Splines are smooth curves whose trajectories are shaped by control points and those points' tangents. Each point also controls a spline's height and thickness attributes which are used to drive the mapping, warping and scattering of images.

This category holds the following sub-categories:

  • Spline tools
    Nodes for generating and ajusting splines, as well as using them for mapping, scattering or warping images.
  • Path tools
    Nodes for generating and adjusting paths, as well as converting them to splines for complex scattering or warping of images.
