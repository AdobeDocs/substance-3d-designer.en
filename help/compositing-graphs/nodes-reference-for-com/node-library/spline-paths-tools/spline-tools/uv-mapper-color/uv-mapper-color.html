In: Spline & Path Tools > Spline Tools

Description

Maps the input color image using the coordinates provided in the UV input.

Note

See also UV Mapper Grayscale.

Input connectors

UV Color
Image coordinates encoded in the red (U) and green (V) channels of a color image.

Input Color
The color image which should be mapped to the coordinates provided in the UV input.

Output connectors

Output Color
The result of mapping the Input image using the input UV coordinates, as a color image.

Parameters

Background Color Float4
The background color of the output image.
The background is visible in the areas of the image where UVs are not defined (I.e., the value is (0, 0, 0, 0)).
