Generates a flat grayscale or color value.

It is a simple node that is used very often as a starting point for adding colors or creating specific values.

Performance optimization

Both of these adjustments lowers the node's computation time and memory footprint:

  • If a grayscale value is needed, make sure to switch the node's color mode to 'Grayscale'.
  • Since the node's output is a flat color, you may use the lowest resolution possible. Set the node's 'Output size' parameter to use the 'Absolute' inheritance method and a resolution of 16x16 pixels.

Parameters

Color mode

Boolean

Toggles between a grayscale and a color output image.

Output color

Float/Float4

Selects the flat color to use in the output image.

 

When using the 'Color' color mode, the Alpha channel is used for opacity where 0 is fully transparent and 1 is fully opaque..
