All other Substance graph nodes in the [Library](../../../help/interface/the-library/the-library.md) are built out of atomic nodes, should you break them down to their lowest level.

## Placing atomic nodes

There are several ways to create atomic nodes in Substance graphs:

### <b>The node palette</b>

![](atomic-bar.png)

The node palette sits in the [Graph View toolbar](../../../help/interface/the-graph-view/the-graph-view.md) and provides easy access to atomic nodes: simply click a node or drag it in the graph.

The palette is toggled using this button: ![](node-palette-toggle.png)

### <b>The node menu</b>

Press *Spacebar* or *Tab* in the Graph View displays a searchable list of nodes and lists all atomic nodes by default.

The search field lets you browse [all the other nodes in the Library](../../../help/compositing-graphs/nodes-reference-for-com/node-library/node-library.md), including [your own content](../../../help/interface/the-library/managing-custom-content/managing-custom-content-and-filters.md) (if added).

### <b>The graph contextual menu</b>

Right-click in the graph and go to the 'Add node' submenu to access a list of atomic nodes.

### <b>The Library</b>

The 'Atomic nodes' category of the [Library](../../../help/interface/the-library/the-library.md) hosts all atomic nodes. They can be dragged and dropped into a Substance graph.

### <b>Keyboard shortcuts</b>

You can define [your own keyboard shortcuts](../../../help/interface/preferences-window/preferences-window.md) to instantly create any node in a graph, including atomic nodes.
