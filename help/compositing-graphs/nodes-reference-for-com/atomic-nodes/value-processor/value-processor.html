Computes a Substance function graph and outputs its result.

It is comparable to a Pixel processor, with the difference that it does not compute a function for every pixel, but rather a single value and makes it available in a Substance graph.

Tip

This node is a good starting point for learning about Substance function graphs.

Also consider that working with this type of graph and performing mathematical operations is mandatory for getting anything out of this node.
