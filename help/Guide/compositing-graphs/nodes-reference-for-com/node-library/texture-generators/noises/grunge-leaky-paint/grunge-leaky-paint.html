Grunge Leaky Paint

In: Texture Generators /Noises

Simple

Description

The Grunge Leaky Paint node generates a grunge map akin to paint dripping through leaks.

Parameters

  • Balance Float
    Adjusts the balance between dark and bright values.
  • Contrast Float
    Adjusts the contrast of the image.
  • Invert Boolean
    Inverts the output of the image, using a 1-x operation.
  • Non Square Expansion Boolean
    Enables compensation of squash and stretch with non-square ratios.
  • Advanced
    • Leak Intensity Float
      Adjusts the density and intensity of the drips.
    • Leak Scale Integer
      Adjusts the scale of the drips separation.
    • Leak Angle Random Float
      Adjusts the maximum angle drips can be randomly rotated to, in number of turns.
    • Leak Crispness Float
      Adjusts the crispness and sharpness of the drips.
