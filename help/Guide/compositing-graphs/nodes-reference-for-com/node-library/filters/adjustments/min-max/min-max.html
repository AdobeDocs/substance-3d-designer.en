Min Max

Min Max

In: Filters/Adjustments

Intermediate

Description

Min Max finds the brightest and darkest values of a Grayscale input and returns them as Values. It's intended as a more granular, manual alternative for Auto Levels, where you expose the Value Inputs of a Levels node and plug the values from Min Max into it.

To use this node with a Levels, you should at least know how to use the Expose Parameter dropdown, as well as the Value Input Tab.

Parameters

No Parameters.
