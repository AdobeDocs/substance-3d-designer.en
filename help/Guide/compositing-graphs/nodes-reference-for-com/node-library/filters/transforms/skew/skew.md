## Description

Skews an input image.

## Parameters

* **Axis**: *Horizontal, Vertical*Choose to skew vertically or horizontally.
* **Amount**: *-1.0 - 1.0*Amount of skew.
* **Align**: *Center, Top Left, Bottom Right*Sets the origin point of the Skew transformation.
